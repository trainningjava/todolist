## :information_source: desafio
* Usuário
  * Cadastrar
  * Atualizar
  * Excluir
  * Listar
* Tarefas
  * Cadastrar
  * Atualizar
  * Excluir
  * Listar

Considere o cadastro com dados básicos:
* Usuário
  * (UUID) id 
  * (String) username 
  * (String) name
  * (String) password
  * (LocalDateTime) createdAt
* Tarefas
  * (UUID) id
  * (String) description
  * (String) title
  * (LocalDateTime) startAt
  * (LocalDateTime) endAt
  * (String) priority
  * (UUID) idUser - id do usuário
  * (LocalDateTime) createdAt
